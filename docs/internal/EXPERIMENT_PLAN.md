## 〇、论文方向

当前所有实验指向：

> **当模型差异度低时，AFP/IVN 无法优于 FedAvg。门控函数 τ/(τ+imp) 在 τ=0.5 时保护下限=0.33，远超安全混合上限 0.1。降低 τ 能否恢复选择性？提高差异度后选择性是否优于平均？这两个问题决定了论文的存在性。**

## 三、教训

1. **E1 的模型比当前所有模型都好** — 必须搞清楚 E1 用了哪组权重。B→code=0.768 vs 当前 0.751。
2. **不对称差异实验（c1m3/c1m5）不应该跑** — 它们测的不是 AFP 的能力，而是"一方太弱时被另一方淹没"。这永远不会有正结果。
3. **队列脚本盲目排 N 个实验** — 应该先跑 1 个，看结果，再决定下一个。
4. **Bug 17 修复的验证用了完整 grid search** — 一个 τ 值就够了。
5. **看到坏结果应该立即停下来想，不要继续跑** — c1m3 的 -0.825 出现时就应该终止队列。

### 4.2 核心结论

**结论 1：AFP/IVN 在低差异度下无法优于 FedAvg。**

8 个实验（包括 E1-E7 和废弃的 E0），跨越 3 种 importance 指标（magnitude, L2 rel, MAS）、6 个 τ 值（0.05-2.0）、4 组模型对（c1m1/c1m3/c1m5 + E1）。无一例外：AFP/IVN 从未产生正净收益。FedAvg α=0.1（10% 均匀混合）是唯一不破坏模型的方法。

**结论 2：门控函数 τ/(τ+imp) 在结构上无法产生差异化结果。**

E7 测试了 τ∈[0.05, 1.0]——gate 范围从 [0.05,0.33] 到 [0.5,0.91]。所有 τ 值下 AFP 输出完全相同（net=-0.149）。因为模型差异太小（mean ΔW < 2%），任何非均匀混合都无法偏离线性插值的损失平面。

**结论 3：IVN 是空操作。** 

全部 8 个实验，IVN 在 1 轮内收敛，ΔV < 0.0001。虚拟谈判没有产生任何有意义的梯度交换。在单一盆地内，Pareto 平稳点 = 任意线性插值点。

**结论 4：线性插值障碍（LMC barrier）是选择性集成的必要条件。**

当前模型对（mean ΔW 0.26%-1.95%）的损失景观在插值路径上近乎平坦。在此条件下：
- 选择性集成 = 盲平均（结论 1）
- 门控参数无影响（结论 2）
- 多轮谈判无信息（结论 3）

**LMC barrier > 0 是 AFP/IVN 有效的必要条件。这是 Phase 0 的核心理论贡献。**

### 4.3 论文叙事

```
标题: When Selective Model Integration Fails: Linear Mode Connectivity
       as a Necessary Condition for Gated Federated Learning

贡献:
1. 在 Pythia-1.4B × VersaPRM (code+medical) 上，系统证明 AFP/IVN
   在低差异度下无法优于 FedAvg（8 个实验，3 种 importance，6 个 τ）
2. 提出 LMC barrier 作为选择性集成的必要条件：
   只有当线性插值存在 loss 障碍时，per-block gate 才有区分度
3. 实验表明 IVN（多轮 Pareto 谈判）在当前条件下等价于单次 AFP
4. gate 函数 τ/(τ+imp) 在模型差异 <2% 时对所有 τ 输出相同结果

基线: FedAvg α=0.1 是当前最优策略（+0.010 到 +0.061）
```
